# SCAI Lucky Loop

A Telegram Web3 Lottery Platform built with React, Solidity, Ethereum and SecureChain AI.

## Features

- MetaMask Wallet Connection
- Lottery Ticket Purchase UI
- Wallet Dashboard
- Referral System UI
- Profile Dashboard
- Recent Winners
- Responsive Design
- Modern Web3 Interface

## Tech Stack

- React.js
- Vite
- React Router DOM
- Ethers.js
- Solidity
- CSS3

## Project Structure

```
client/
server/
contracts/
docs/
```

## Installation

```bash
git clone <repository-url>
cd client
npm install
npm run dev
```
